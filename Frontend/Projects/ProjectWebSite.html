Projekt: Webseite

Beschreibung: Meine persönliche Webseite, die ich mit HTML, CSS und JavaScript erstellt habe. Sie dient als Portfolio und Informationsquelle über mich und meine Projekte.

Darüber hinaus habe ich,um mein Können darin zu schärfen, ein Java Backend mit Springboot sowie eine Anbindung an eine postgresql-Datenbank entworfen.

Für das Potential der fortlaufenden Entwicklung und Veröffentlichen habe ich ebenfalls Docker-Container und eine CI-CD Pipeline auf Github eingerichtet.

Die Kenntnisse hierfür, konnte ich aus einem Praktikum zum Thema Software Engineering an der Uni Leipzig entnehmen, bei dem unter Betreuung von Adesso SE eine solche Seite eingerichtet wurde.
